# tp-hackaton (frontend)

⚠️ Note hébergement Render (backend) :

Ce projet utilise un backend hébergé gratuitement sur Render. Le service Render gratuit suspend le backend après 15 minutes d’inactivité. Lors de la première visite, il peut donc y avoir un délai de 10 à 30 secondes avant que le site ne réponde pleinement. Après ce délai, tout fonctionne normalement.

Si la page met du temps à charger, patientez quelques secondes puis rafraîchissez.
